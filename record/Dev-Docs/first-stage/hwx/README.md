本目录记录了黄雯萱在初赛阶段的开发流程及学习笔记。   
   
项目在初赛的开发流程可按时间分为三个阶段，分别使用三个目录记录了详细的开发流程：   
   
* 3.27 - 4.4   
   
  确认选题阶段。此目录下记录了我们确认选题和基本思路形成的会议纪要。   
   
* 4.5 - 5.6   
   
  调研阶段。此目录下记录了我们在确认选题之后进行的调研工作。   
   
* 5.7 - 初赛结束   
   
  项目搭建阶段。此目录下记录了我们在项目开发阶段的内部文档。   
   
除此之外的doc目录记录了初赛文档产出的中间流程，包含原始数据记录以及项目文档初稿。
